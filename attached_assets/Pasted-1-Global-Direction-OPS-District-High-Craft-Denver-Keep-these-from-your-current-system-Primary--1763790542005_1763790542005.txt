1️⃣ Global Direction — OPS District, High-Craft Denver
Keep these from your current system:
Primary blue brand
Real project photos
Section spacing + container system (max-w-7xl mx-auto px-6)
BBB + testimonials + contact CTAs
Add/shift:
Warm wood accent to complement blue
--accent-wood: #C8945C; (muted caramel / wood tone)
Softer background, less pure white
#F5F4F2 for most “light” sections
More refined type pairing
Headings: a slightly more “crafted” serif or display (e.g. Playfair Display, Cormorant Garamond)
Body: keep your clean sans (e.g. Inter, System UI)
In index.css (or global styles), you can do something like:
:root {
  --brand-blue: 233 48% 45%; /* existing HSL */
  --brand-blue-soft: 226 40% 18%;
  --accent-wood: 32 52% 57%;
  --bg-soft: #F5F4F2;
}

body {
  @apply bg-[#060713] text-slate-100; /* for very dark hero routes */
}

/* Light sections */
.section-light {
  background-color: var(--bg-soft);
}

/* Headings */
h1, .heading-xl {
  font-family: "Playfair Display", system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
}

/* Buttons */
.btn-primary {
  @apply inline-flex items-center justify-center rounded-full px-6 py-3 text-sm font-medium tracking-wide;
  background: linear-gradient(135deg, hsl(var(--brand-blue)), hsl(var(--brand-blue-soft)));
  box-shadow: 0 12px 30px rgba(15, 23, 42, 0.45);
}
.btn-primary:hover {
  filter: brightness(1.06);
}
.btn-outline-light {
  @apply inline-flex items-center justify-center rounded-full px-6 py-3 text-sm font-medium border border-white/30 text-white/90;
  backdrop-filter: blur(8px);
}
Cyborg can adapt that to however your Tailwind/utility setup is wired.

4️⃣ PortfolioSection — Let the Work Sell It
You already have:
40+ images
Filtering
Lightbox
We just want to make the cards themselves more cinematic.
Suggestions for the project card component:
Remove heavy borders; use soft rounded corners + subtle shadow.
On hover:
slight scale up (scale-105)
dark overlay with project name + location in bottom left
Add a small “Before / After” tag if available.
Example card styling guideline:
export function ProjectCard({ image, title, location }: Props) {
  return (
    <div className="group relative overflow-hidden rounded-2xl bg-slate-900/60 border border-slate-800/80 shadow-[0_18px_45px_rgba(15,23,42,0.85)]">
      <div className="aspect-[4/3] overflow-hidden">
        <img
          src={image}
          alt={title}
          className="h-full w-full object-cover transition-transform duration-500 group-hover:scale-105"
        />
      </div>

      {/* gradient overlay */}
      <div className="pointer-events-none absolute inset-0 bg-gradient-to-t from-slate-950/90 via-slate-950/20 to-transparent opacity-80" />

      {/* text on hover */}
      <div className="absolute inset-x-4 bottom-4 flex flex-col gap-1">
        <p className="text-sm font-semibold text-slate-50">{title}</p>
        <p className="text-xs text-slate-300/90">{location}</p>
      </div>
    </div>
  );
}
This makes the work your hero, instead of UI chrome.
